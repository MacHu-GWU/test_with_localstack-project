Install
------------------------------------------------------------------------------

``test_with_localstack-project`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install test_with_localstack-project

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade test_with_localstack-project
